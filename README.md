## About App
App has 2 account types. One for admin accounts and one for normal users. 

### `Admin Permisions`
If we have admin permission we can:
- Create new user
- Delete user
- Edit info about user
- Send Message (In progress)

### `User Permisions`
If we have user permission we can:
- Read Message
